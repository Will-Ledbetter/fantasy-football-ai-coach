# 🚀 Super Easy Team Connection Guide

## Overview

We've made connecting your fantasy teams as simple as possible. Here are your options, ranked from easiest to most technical:

## 🥇 Option 1: One-Click Sleeper Connection (Easiest)

**Best for:** Sleeper users who want zero hassle

1. **Click "Connect with Sleeper"** - Opens Sleeper OAuth
2. **Authorize the app** - Grant read-only access to your leagues
3. **Select your league** - Pick from your active leagues
4. **Done!** - No IDs, tokens, or cookies needed

**Time:** 30 seconds

## 🥈 Option 2: URL Auto-Detection (Very Easy)

**Best for:** All platforms when you have your league URL handy

1. **Copy your league URL** from your browser
   - Sleeper: `https://sleeper.app/leagues/123456789`
   - ESPN: `https://fantasy.espn.com/football/league?leagueId=123456`
2. **Paste it in our setup form** - We auto-extract everything
3. **Add credentials if needed** (ESPN private leagues only)
4. **Test connection** - We verify it works
5. **Done!**

**Time:** 2 minutes

## 🥉 Option 3: Browser Extension Helper (Easy for ESPN)

**Best for:** ESPN users who want automated cookie extraction

1. **Install our Chrome extension** (coming soon)
2. **Visit your ESPN league page**
3. **Click the extension icon** - Auto-extracts cookies
4. **Copy to setup form** - One-click copy
5. **Done!**

**Time:** 3 minutes

## 🔧 Option 4: Manual Setup (Traditional)

**Best for:** Advanced users or troubleshooting

### Sleeper Manual Setup
1. Get League ID from URL: `sleeper.app/leagues/YOUR_LEAGUE_ID`
2. Get User ID: Sleeper App → Settings → Account → User ID
3. Enter both in our setup form

### ESPN Manual Setup
1. Get League ID from URL: `fantasy.espn.com/football/league?leagueId=YOUR_ID`
2. Get cookies (private leagues only):
   - Press F12 → Application → Cookies → fantasy.espn.com
   - Copy `espn_s2` value
   - Copy `SWID` value (include { } brackets)
3. Enter all in our setup form

**Time:** 5-10 minutes

## 🆘 Need Help?

### Common Issues

**"League not found"**
- Check if league is public/private
- Verify you're using the correct League ID
- For ESPN: Make sure cookies are fresh (re-login if needed)

**"User not found in league"**
- Double-check your User ID
- Make sure you're actually in this league
- For Sleeper: User ID is different from username

**"Connection test failed"**
- Try refreshing your browser cookies
- Check if the fantasy platform is down
- Contact support with your error message
